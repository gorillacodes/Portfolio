Tinder Clone

This was a project I started when I first learned Reactjs. I was the sole creator/developer for this project.

It is the clone of the most popular dating app Tinder, made by using React.js and other front-end Technologies.

Basically, it is just the UI part of the actual tinder app and further i am planing to do make it a real world working app by using MERN stack Technologies.

Also, in near furture i will deploy this, make it online and i've planned to add couple of features of my own that i think should be present in a dating app.

Technologies:
  • - React JS
  • - Redux
  • - Context API
  • - React Hooks
  • - Semantic UI
  • - CSS
  • - Javscript
